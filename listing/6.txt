package main
 
import (
  "fmt"
)
 
func main() {
  var s = []string{"1", "2", "3"}
  modifySlice(s)
  fmt.Println(s)
}
 
func modifySlice(i []string) {
  i[0] = "3"
  i = append(i, "4")
  i[1] = "5"
  i = append(i, "6")
}

Программа выведет [3 2 3]. Сначала создаётся слайс с указателем на базовый массив из трёх элементов len=3 и cap=3. 
Затем он передаётся по значению в функцию modifySlice, а именно в новую локальную переменную i копируются указатель на массив, len и cap. 
Затем меняется нулевой элемент слайса, и это изменение повлияет на слайс, который был создан в основной функции, потому что оба слайса имеют указатель на один
и тот же базовый массив. Но затем при выполнении функции append создаётся новый слайс с новым элементом 4, но при этом также создаётся новый массив с cap=6,
потому что прошлая cap была превышена, и новый слайс будет иметь указатель на новый базовый массив, присваиваем переменной i новый слайс.
И следующее изменение первого элемента на значение 5 не повлияет на слайс, созданный в основной функции, потому что теперь слайс i указывает на другой базовый массив.
Также следующая функция append создаст новый слайс с len=5 и присвоит его переменной i. Это опять не произведёт изменения на слайс в основной функции
